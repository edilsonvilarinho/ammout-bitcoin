# bitcoin

Projeto que consome API https://blockchain.info/ticker

 - MVC
 - Provider
 
# Estrutura do projeto

  - Padrão MVC
    - Funcionalidade 
      - controller
      - model
      - imagens
      - service
      - view
   - Provider como controle de estado concentrado no AppWidget para ser utilizado em qualquer parte do app pelo o comsumer.
